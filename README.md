# InteractoBot

# Requirements

**Unity:** 6000.0.36f1 LTS

**Newtonsoft Json** for json file parsing

1. Open Unity Editor
2. Go to **Window** -> **Package Manager**
3. Click the "+" symbol, then "Install package by name"
4. Enter `com.unity.nuget.newtonsoft-json`, then click "Install"
